Aspect-Based Sentiment Analysis

Aspect-based sentiment analysis (ABSA) is a text

analysis technique that categorizes data by aspect

and identifies the sentiment attributed to each one

Our ABSA model can provide

Using CNN, LSTM, Bi-LSTM, PhoBERT and BERT-multilingual to handle this task. Using many methods to training models. Handle imbalanced dataset with class weights. Evaluating models via F1-macro and F1-micro. Using Flask to deploy best model to AWS
